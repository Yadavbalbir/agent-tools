# Security Policy

## Supported Versions

This repository is actively maintained on the default branch.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Instead, report privately to the repository maintainers with:
- Description of the issue
- Impact assessment
- Reproduction steps
- Any suggested mitigation

We aim to acknowledge reports within 72 hours and provide a remediation plan promptly.

## Scope Notes

This project contains prompt/skill definitions and templates. Security risks may include:
- Prompt injection vectors in skill instructions
- Unsafe command suggestions
- Leakage of sensitive data in examples

Contributors should avoid embedding secrets, tokens, or private data in any file.
