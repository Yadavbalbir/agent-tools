# Document Checklist — Microsoft India ITR

Tick each item. Ask the user to drop files into the workspace. Essentials block computation; the rest refine it.

## A. Salary (Microsoft) — essential
- [ ] Form 16 Part A (TDS summary)
- [ ] Form 16 Part B (salary + deductions)
- [ ] Form 12BA (perquisites: RSU vest value + ESPP discount)
- [ ] Payslips Apr–Mar (March payslip confirms YTD)

## B. Indian broker capital gains — essential if traded
- [ ] Tax P&L / Capital Gains statement (stocks + mutual funds) for the FY
- [ ] Segments traded? delivery (STCG/LTCG) / intraday (speculative) / F&O
- [ ] F&O or intraday ledger / contract notes (for turnover)

## C. Microsoft RSU/ESPP via Fidelity — essential (foreign)
- [ ] Fidelity Year-End Investment Report (1 Jan–31 Dec)
- [ ] Form 1042-S (US tax withheld on dividends)
- [ ] Holdings on 31 Dec, cost basis, peak value
- [ ] RSU vest dates + quantities; ESPP purchase dates + quantities
- [ ] First acquisition (earliest vest) date

## D. Other income — essential
- [ ] Bank interest certificates (savings + FD) — ALL banks
- [ ] Dividend statements — Indian shares
- [ ] Rental income details (if any)

## E. Validation — essential (download from incometax.gov.in)
- [ ] AIS (Annual Information Statement)
- [ ] TIS (Taxpayer Information Summary)
- [ ] Form 26AS

## F. Optional / prior year
- [ ] Last year's ITR (JSON / acknowledgement) for a year-over-year baseline
- [ ] Last year's Intimation u/s 143(1) — to read Sec 244A refund interest (taxable this year)

## Not needed in the New Regime
- 80C / 80D / HRA / LTA / 80TTA proofs (all disallowed)
- Exception: 80CCD(2) employer NPS, only if the CTC includes an NPS contribution
